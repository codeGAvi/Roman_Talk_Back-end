# RomanTalk Backend

Spring Boot backend for RomanTalk.

## Features

* Translate 20+ languages into Roman script
* Gemini AI Integration
* PostgreSQL (Neon)
* Redis (Upstash)
* Razorpay Payments
* JWT Authentication

## Tech Stack

* Java 21
* Spring Boot 3
* Spring AI
* PostgreSQL
* Redis
* Docker

## Status

Backend service running on port 8080.
